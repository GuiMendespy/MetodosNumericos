# 📊 Introduction to NumPy – DataCamp

Este repositório contém meus estudos e exercícios práticos realizados durante o curso **“Introduction to NumPy”** da DataCamp.

Ao longo do curso, desenvolvi uma base sólida em computação numérica com Python utilizando a biblioteca NumPy, com foco em manipulação eficiente de arrays, operações vetorizadas e otimização de desempenho.

## 🧠 Conteúdos Abordados

* Criação e manipulação de arrays 1D, 2D e 3D
* Métodos de agregação (`sum`, `min`, `max`, `mean`, `cumsum`)
* Uso dos argumentos `axis` e `keepdims`
* Operações vetorizadas para melhorar desempenho
* Operações matemáticas e lógicas elemento a elemento
* Regras de broadcasting e compatibilidade entre dimensões
* Representação de imagens utilizando arrays RGB
* Salvamento e carregamento de arrays no formato `.npy`
* Transformações de arrays: `reshape`, `flatten`, `transpose`
* Técnicas de manipulação: `flip`, `split`, `stack`
* Aplicações práticas com visualização utilizando Matplotlib

## 🚀 Habilidades Desenvolvidas

* Escrita de código numérico limpo e eficiente
* Substituição de loops por operações vetorizadas
* Compreensão de dimensões e estrutura de arrays multidimensionais
* Manipulação de dados em formato matricial
* Aplicação de NumPy em problemas práticos e análise de dados

Este curso fortaleceu minha base em computação científica com Python e serviu como preparação para tópicos mais avançados, como análise de dados com pandas, visualização de dados e aprendizado de máquina.
